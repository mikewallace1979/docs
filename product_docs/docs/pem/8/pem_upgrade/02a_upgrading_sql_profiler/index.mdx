title: "Upgrading SQL Profiler"
---

The process of upgrading SQL Profiler is platform-specific. You can update SQL Profiler on a Linux platform by using the native packages and on a Windows platform by using the graphical installer available for Windows. 

This section provides detailed information about upgrading the SQL Profiler:

-   **Upgrading a SQL Profiler Installation on Windows** - Provides information about upgrading your SQL Profiler from one major version to another on the Windows platform.

!!! Note
    - Prior to PEM 7.8 release, SQL Profiler could be installed on Linux either by using the     
      graphical installer or by using the RPMs. 
    - From PEM version 7.8 onwards, SQL Profiler graphical installers for Linux are discontinued. To upgrade a 
      SQL Profiler on a Linux host from any lower version to PEM 7.9 or higher versions, you must use native packages.

Links to SQL Profiler installers and RPMs are available at the [EDB website](https://www.enterprisedb.com/downloads/edb-postgres-enterprise-manager).
