| Operating systems                   | JDK versions                                                                                                          |
| --------------------------------------- | ------------------------------------------------------------------------------------------------------------------------- |
| `CentOS 7 and Rocky Linux/AlmaLinux 8`<br />                  | Red Hat OpenJDK 8     <br />                                                                                              |
| `SLES 12`<br />                         | Red Hat OpenJDK 8     <br />                                                                                              |
| `PPCLE RHEL 8`                           | Red Hat OpenJDK 8 |
| `PPCLE RHEL 7`                           | Red Hat OpenJDK 8     <br />                                                                           |
| `RHEL 7 and 8`<br />             |  -   Red Hat OpenJDK 8     <br /> <br /> -   Oracle JDK 8     <br />                                                      |
| `Windows 2012 R2, 2016, and 2019`<br /> | Red Hat OpenJDK 8     <br />                                                                                       |
| `Debian 10`                             | Red Hat OpenJDK 11<br /><br />                                                                          |
| `Ubuntu 18 and 20`                      | OpenJDK 11<br /><br />                                                          |               
| `Debian 9`                              | OpenJDK 8<br /><br />                                                                   |               

See [Product Compatibility](https://www.enterprisedb.com/platform-compatibility#eprs) for more information.
